# Decision Log Template

Each new decision/topic gets its own file in this folder, named
`YYYY-MM-DD-short-topic.md` (e.g. `2026-05-10-labeling.md`).
Copy the structure below.

Keep entries focused: what we decided, why, what we rejected. Don't restate
things already settled — link to earlier dated files instead.

---

**Why:** <reasoning>
